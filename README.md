# C.I.A — Conteúdo Infinito com IA

Landing page em Next.js 14 + TypeScript + Tailwind CSS + Framer Motion.

## 🚀 Deploy (GitHub + Vercel)

1. Suba esta pasta no GitHub
2. Entre em [vercel.com](https://vercel.com) → New Project → importe o repo
3. Vercel detecta Next.js automaticamente → clique Deploy
4. Em Settings > Domains → adicione seu domínio da Hostinger

## 🖼️ Imagens

Coloque suas imagens em `/public/assets/images/`:
- logo_oficial.jpeg
- mockup1.jpeg até mockup9.jpeg

## 💻 Rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
